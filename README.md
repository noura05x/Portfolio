##  Installation

Clone the repository:

```bash
git clone https://github.com/noura05x/Portfolio.git
```

Navigate to the project folder:

```bash
cd Portfolio
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open:

```text
http://localhost:3000
```
